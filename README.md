# IW Frutas y Verduras

Aplicación móvil desarrollada en Flutter para la venta de frutas y verduras.

## Funciones

- Inicio de sesión
- Catálogo de productos
- Búsqueda de productos
- Carrito de compras
- Información de la tienda

## Tecnologías utilizadas

- Flutter
- Dart
  
## Instalación

1. Descargar o clonar el repositorio.
2. Abrir el archivo main.dart en DartPad o en un entorno Flutter.
3. Ejecutar la aplicación.

## Uso

1. Iniciar sesión.
2. Explorar el catálogo de frutas y verduras.
3. Buscar productos.
4. Agregar productos al carrito.
5. Finalizar la compra.
